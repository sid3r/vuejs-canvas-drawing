# A simple vuejs drawing app
Made with Vuejs, Konva and Tailwind css

![name-of-you-image](public/screen.png)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Lint files
```
yarn lint
```
